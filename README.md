# Reef.js Boilerplate Code
---
This is some reef.js and css boilerplate code to create projects faster. It includes a colour scheme template and some useful premade classes. 

To use it press "Use this template" and press create a new repository to create a new repository with this code or select the template when creating a new repository. 

Reef.js is an antiframweork designed to create state based uis without complicated syntax. You can find the reef.js documentation here: https://reef.gomakethings.com/
